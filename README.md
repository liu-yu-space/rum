# rum-request-upload-manager

一个基于 TypeScript 的请求管理与上传管理库，支持未来扩展 Socket 和 SSE。

## 特性

- 请求管理
- 上传管理
- 未来支持 Socket 和 SSE
- TypeScript 支持
- 适用于 Node.js 和浏览器

## 快速开始

```bash
pnpm install rum-request-upload-manager
```

## 使用

```ts
import { RequestManager, UploadManager } from "rum-request-upload-manager";

// 示例代码
```

## 开发

- 构建：`pnpm build`
- 测试：`pnpm test`
